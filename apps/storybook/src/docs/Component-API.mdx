### Combobox

<Canvas of={ComboboxStories.Controlled} />
<Controls of={ComboboxStories.WithCustomEmptyMessage} />

| Prop | Type | Notes |
| --- | --- | --- |
| `options` | `ComboboxOption[]` | Supports disabled entries and keyword filtering. |
| `value` / `defaultValue` | `string` | Controlled and uncontrolled modes are both available. |
| `onValueChange` | `(value: string) => void` | Fires when user confirms option by click or Enter. |
| `emptyMessage` | `ReactNode` | Custom no-result content for search miss scenarios. |

### DatePicker

<Canvas of={DatePickerStories.Controlled} />
<Controls of={DatePickerStories.InvalidState} />

| Prop | Type | Notes |
| --- | --- | --- |
| `value` / `defaultValue` | `string` | Uses native `YYYY-MM-DD` value format. |
| `onValueChange` | `(value: string) => void` | Emits selected date string from native date input. |
| `min` / `max` | `string` | Native date range constraints. |
| `invalid` | `boolean` | Adds `aria-invalid` to communicate validation state. |

### Table

<Canvas of={TableStories.ReleaseChecklist} />

| Prop | Type | Notes |
| --- | --- | --- |
| `columns` | `TableColumn<T>[]` | `sortable` + optional `sortAccessor` for stable sorting behavior. |
| `data` | `T[]` | Empty arrays render `emptyContent`. |
| `defaultSortKey` / `defaultSortDirection` | `string` / `'asc' \| 'desc'` | Sets initial `aria-sort` and data order. |
| `onSortChange` | `(key, direction) => void` | Useful for telemetry and analytics. |

<Canvas of={TableStories.SortTelemetry} />

### Pagination

<Canvas of={PaginationStories.Controlled} />
<Controls of={PaginationStories.CompactRange} />

| Prop | Type | Notes |
| --- | --- | --- |
| `page` | `number` | Current page index (1-based). |
| `pageCount` | `number` | Total number of pages; values `<= 1` render no UI. |
| `onPageChange` | `(page: number) => void` | Called for navigation intent. |
| `showFirstLast` | `boolean` | Toggle first/last jump buttons. |
| `siblingCount` / `boundaryCount` | `number` | Controls visible number tokens before ellipsis. |
