### Dropdown

<Canvas of={DropdownStories.Default} />
<Controls of={DropdownStories.Default} />

| --- | --- | --- |

| menu naming | trigger-linked `aria-labelledby` | Menu inherits an accessible name from trigger content for screen-reader context. |
| trigger relation | conditional `aria-controls` | Trigger points to menu ID only when expanded. |
| keyboard behavior | Arrow / Home / End / Escape / Tab | Roving-focus menu navigation with deterministic close policy. |
