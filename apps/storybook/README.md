# Aurora UI Storybook

## Start
```bash
pnpm --filter @aurora-ui/storybook-app storybook
```

## Build
```bash
pnpm --filter @aurora-ui/storybook-app build-storybook
```

## Notes
- Theme switcher is available from Storybook toolbar (`core-light`, `core-dark`, `glass`, `neo-brutal`).
- Stories are focused on real library components rather than Storybook template examples.
